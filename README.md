# Web Marketing Labs

Este repositório é o meu laboratório digital, onde documento experimentos práticos na interseção entre **Design de Interfaces (UI)**, **Experiência do Usuário (UX)** e **Estratégia de Marketing**.

## 🎯 Objetivo
O propósito deste espaço é iterar sobre conceitos de design, testar fluxos de navegação e implementar landing pages focadas em conversão.

## 🛠 Tech Stack
* **Design & Prototipagem:** Figma
* **Front-end:** HTML5, CSS3, JavaScript
* **Metodologias:** Mobile-first, WCAG 2.1 (Acessibilidade)
